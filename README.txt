Данное приложение представляет из себя интерфейс, при помощи которого удобно постить объявления по сообществам vk.com.
***********************************************************************************************************************

1. Открываете MySQL Workbench и логинитесь под своим коннекшином(удостоверьтесь, что у вашего пользователя есть права на чтение и изменение, а лучше всего залогиньтесь под root-пользователем).В данной инструкции я буду логиниться под

пользователем - root
адресом хоста - 127.0.0.1 (localhost)
паролем - tsvetsv88

2. Открываете файл db_scripts\vk_ad_posting.sql в MySQL Workbench и весь его запускаете. У вас создастся база messages_tmp и таблица в ней communities(если вас не устраивают название базы и таблицы, перед запуском можете поменять их в данном файле при помощи комбинации клавиш CTRL + H - в данной же инструкции названия остаются теми же). Сущность таблицы messages_tmp.communities представляет из себя набор vk-сообществ, по которым мы хотим постить наши объявления.

3. Логинитесь в vk.com под своим логином и устанавливаете к себе приложение по адресу: https://vk.com/app5094747_334917471

4. Заходите в корень папки приложения через cmd, собираете и запускаете его следующими командами:

	npm install

	grunt 

	node server communities messages_tmp 127.0.0.1 root tsvetsv88   (здесь | communities messages_tmp 127.0.0.1 root tsvetsv88 | являются входными параметрами, которые описаны в пунктах 1 и 2)

5. Ооткрывате ваш браузер по адресу https://127.0.0.1:8000 - если браузер говорит что соединение незащищено, нажимаете кнопки типа "Advanced" или "Все равно перейти"(Удостоверьтесь что порт 8000 не занят каким-нибудь другим процессом на вашем компе, иначе придется его освободить, тк данное приложение работает на нем)

6. Логинитесь в vk.com под своим логином и заходите по адресу: https://vk.com/app5094747_334917471 - открываете тем самым vk-приложение из пункта 3.

7. Параметры приложения для обращения к базе данных вбиты по умолчанию, вам нужно только выбрать в поле "Всего отправок" число отличное от нуля - оно создано, чтобы случайно не отослать слишком много сообщений в сутки через один логин, из-за чего могут забанить. Далее жмете кнопку "Получить сообщества" и получаете список сообществ, в которые можете отправить свои сообщения при помощи кнопки "Отправить" или вообще удалить сообщесто из базы при помощи кнопки "Удалить".
